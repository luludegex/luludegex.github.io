Mes conaissances en HTML et en CSS !




Voici ce que j'ai pu apprendre grâce à la fabrication de ce splendide site web :

  • J'ai appri à comprendre ce qu'était le Html et en quoi ca consistait:
  • Le Html sert à créer un site web

    • (on peut tout de même constater qu'un site web uniquement basé sur le Html, ne donne pas grand chose en terme d'héstétisme...
    ---> voici donc cette même page sans CSS(ou presque la même...).

  • Le principe est d'utilliser des balises et de les agencer de la manière souhaitée pour y afficher le texte, les images ou encore des liens.


  • Le CSS lui, est nécessaire pour obtenir un résultat convenable :

    • On le met en lien avec la page HTML, et on y ajoute des indications sur la mise en page souhaitée.
    • On peut donc sans grand risque dire qu'une page de CSS associée à la page HTML est en fait une page de style pour le site.
      Il y a donc énormément de styles possibles, malléables aux goûts de chacun!
